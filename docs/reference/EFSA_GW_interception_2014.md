# Subset of EFSA crop interception default values for groundwater modelling

Subset of EFSA crop interception default values for groundwater
modelling

## Usage

``` r
EFSA_GW_interception_2014
```

## Format

A matrix containing interception values, currently only for some
selected crops

## Source

European Food Safety Authority (2014) EFSA Guidance Document for
evaluating laboratory and field dissipation studies to obtain DegT50
values of active substances of plant protection products and
transformation products of these active substances in soil. *EFSA
Journal* **12**(5):3662, 37 pp., doi:10.2903/j.efsa.2014.3662
